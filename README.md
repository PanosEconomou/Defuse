# defuse

A utility to solve for the simple defect lines of a 2D Rational CFT.



## Install





## Dependencies

You will need a couple of tools to run defuse (if you use the [installation script](https://xkcd.com/3201/) you don't actually need to download them manually)

1. [4ti2](https://4ti2.github.io): we use `zsolve` and `hilbert` to calculate a Hilbert basis for a set of inequalities
2. [GAP](https://www.gap-system.org): A computer algebra system that spits out character tables like there is no tomorrow to help identify invertible defects.
